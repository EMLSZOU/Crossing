<center>函数</center>
============
一个函数就是将一些逻辑、流程、数据、语句集合在一起的部件，这样就能反复运行。
函数的作用：代码复用、减少冗余，并且也是一种设计手段————将复杂的系统分解为可管理的部件。
函数涉及的语法：

- 函数定义  def function(a, b=1, *c):
  def 是一个可执行语句。创建函数对象，并且将这个函数对象赋值给函数名。
- 返回值  return
- 变量作用域  global  nonlocal
- 迭代返回  yield
- 匿名函数  lambda
- 调用 function()
